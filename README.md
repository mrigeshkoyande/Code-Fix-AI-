# Code-Fix-AI - Syntax Error Checker 🔍

A powerful web-based syntax error checker that helps developers identify and fix code errors in real-time. Built with Flask, JavaScript, and modern web technologies.

## Features ✨

- **Multi-Language Support**: Check syntax errors for Python, JavaScript, Java, C++, C#, and more
- **Real-Time Validation**: Instant feedback on code syntax
- **Clean Interface**: User-friendly web interface with syntax highlighting
- **Detailed Error Messages**: Clear error descriptions with line numbers
- **No Installation Required**: Use directly in your browser

## Technologies Used 🛠️

- **Backend**: Python Flask
- **Frontend**: HTML5, CSS3, JavaScript
- **Language Support**: Python, JavaScript, Java, C++, C#, etc.

## Installation & Setup 🚀

### Prerequisites
- Python 3.7 or higher
- Flask (`pip install flask`)
- Language compilers/interpreters (Python, Node.js, Java, etc.)

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/mrigeshkoyande/Code-Fix-AI-.git
   cd Code-Fix-AI
   ```

2. **Install Flask**
   ```bash
   pip install flask
   ```

3. **Run the application**
   ```bash
   python app.py
   ```

4. **Open in browser**
   Navigate to `http://localhost:5000`

## Usage 💻

1. Select a programming language from the dropdown
2. Paste your code in the code editor
3. Click "Check Syntax" button
4. View the results instantly:
   - ✅ "No syntax errors found!" if code is clean
   - ❌ Detailed error message if issues are found

## How It Works ⚙️

1. **Code Submission**: User submits code through the web interface
2. **Temp File Creation**: Code is written to a temporary file
3. **Syntax Check**: Language-specific syntax checker runs on the code
4. **Error Detection**: Captures and formats error output
5. **User Feedback**: Displays results in a clean, readable format
